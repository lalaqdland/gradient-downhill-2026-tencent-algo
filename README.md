# gradient-downhill-2026-tencent-algo

用于整理 2026 腾讯广告/算法相关比赛资料、过程记录与最终交付物的仓库。

## 建议目录

- `briefs/`: 赛题说明、主办方文档、规则更新
- `research/`: 调研资料、参考链接、行业案例
- `data/`: 可公开提交的数据样例、字段说明、处理中间产物
- `experiments/`: 实验记录、方案草稿、代码原型
- `deliverables/`: 最终提交材料、答辩稿、海报等
- `assets/`: 图片、图表、素材文件
- `notes/`: 会议纪要、任务拆分、阶段复盘

## 使用建议

1. 原始赛题资料优先放在 `briefs/`，并尽量保留来源和日期。
2. 过程中的想法、尝试和结论分别沉淀到 `research/`、`experiments/`、`notes/`。
3. 准备提交时，把对外版本统一收敛到 `deliverables/`。
